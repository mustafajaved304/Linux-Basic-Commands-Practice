# Linux Basic Commands Practice

## Overview
This is a personal project focused on learning and practicing basic Linux commands using Kali Linux.

## Features
- File and directory navigation (ls, cd, pwd)
- File management (cp, mv, rm, mkdir)
- File viewing commands (cat, less, head, tail)
- System information commands (uname, whoami, uptime)
- Disk and memory monitoring (df, du, free)
- Search and help commands (find, locate, man)

## Environment
- Kali Linux
- Bash Shell
- Linux Terminal

## Project Files
- Linux_Basic_Commands_Report.docx

## Learning Outcome
This project helped in building a strong foundation in Linux command-line usage, which is essential for cybersecurity, DevOps, and system administration roles.
